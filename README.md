# About
This is a modern and responsive project management tool designed to help teams organize tasks, track progress, and collaborate efficiently in one place.

# Tech Stack
- Frontend/Backend: Next.js
- Styling: Tailwind CSS
- Database: MongoDB

# Features
- Add, edit, and delete tasks
- Organize tasks by status or progress
- Assign users to tasks
- User-friendly and responsive interface
- Secure data storage with MongoD
- User Dashboard
