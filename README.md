# About
This is a modern and responsive project management tool to help teams organize tasks, track progress, and collaborate efficiently in one place.

# Tech Stack
Frontend: Next.js
Styling: Tailwind CSS
Database: MongoDB

# Features
Add, edit, and delete tasks
Organize tasks by status or progress
Add assignee to tasks
User-friendly and responsive UI
Secure data storage using MongoDB
Fast and optimized performance with Next.js
